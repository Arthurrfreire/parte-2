# Hero de Landing Page - Curso Python com IA

Ferramentas de IA usadas: ChatGPT/Codex para acelerar estrutura, copy visual, ajustes de UI e revisão técnica.
A IA ajudou mais na criação da base Next.js, composição da hero, organização de SEO técnico e refinamentos responsivos.
Também usei IA para comparar referências visuais e transformar os requisitos em uma interface pronta para avaliação.
Eu ajustei manualmente tipografia, contraste, espaçamentos, hierarquia visual e estados dos botões.
Esses ajustes foram feitos para aproximar a tela da identidade da Asimov, melhorar legibilidade e deixar a entrega mais polida.
Também revisei manualmente o código para manter a implementação simples, sem dependências desnecessárias.
Validei a entrega com build de produção usando `npm run build`.
