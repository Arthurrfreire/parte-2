# Hero de Landing Page - Curso Python com IA

Entrega criada com Next.js, React e Tailwind CSS, com foco em SEO tecnico, responsividade e acessibilidade.

**Ferramentas de IA usadas:** ChatGPT/Codex para acelerar estrutura, composicao visual, ajustes de UI e revisao tecnica.
**Onde a IA ajudou mais:** base do projeto, hero section, metadata SEO, animacao do painel de codigo e refinamentos responsivos.
**Ajustes manuais:** tipografia, contraste, espacamentos, paleta, hierarquia visual, hover/focus e polimento dos componentes.
**Motivo dos ajustes:** melhorar legibilidade, consistencia visual, acessibilidade e aproximar a entrega do padrao esperado para avaliacao.

Validacao final: `npm run build`.
